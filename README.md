 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot built with Python. The chatbot responds to predefined user inputs using `if-elif` conditions and keyword matching. It is designed for beginners who want to understand the basics of chatbot development without using machine learning.

 Features

- Responds to greetings (Hi, Hello, Hey)
- Answers basic questions
- Tells its name
- Responds to "How are you?"
- Gives the current time (optional)
- Simple and beginner-friendly code
- Runs in the terminal

 Technologies Used

- Python 3
- Datetime module (optional)

 Project Structure

```
Rule_Based_AI_Chatbot/

```

 How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/Rule_Based_AI_Chatbot.git
```

2. Open the project folder:
```bash
cd Rule_Based_AI_Chatbot
```

3. Run the chatbot:
```bash
python main.py
```

Example Conversation

```
You: Hi
Bot: Hello! How can I help you?

You: What is your name?
Bot: I am a Rule-Based AI Chatbot.

You: How are you?
Bot: I'm doing great! Thanks for asking.

You: Bye
Bot: Goodbye! Have a great day.
```
